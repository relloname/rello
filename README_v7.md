# Marketing Report v7 - 사용 가이드

## 🎉 완성!

모든 요구사항이 성공적으로 구현된 **marketing_report_v7.html** 파일이 생성되었습니다.

## 📋 새로운 기능

### 1. 🛍️ 상품 분석 탭
- 상품별 성과 데이터 분석
- 상품 이미지 표시
- 필터: 캠페인, 광고그룹, 디바이스
- ROAS 하이라이트 및 정렬
- 증감율 표시 (기존 디자인과 동일)

### 2. 📤 개선된 업로드 페이지
- 4개의 드래그&드롭 영역:
  1. RAW (일/주별) - CSV/XLSX
  2. RAW2 (검색어) - CSV/XLSX  
  3. 소재 (상품 성과) - CSV
  4. 상품 TSV - TSV
- 업로드 완료 시 체크 표시 및 색상 변화

### 3. 🔝 TOP10 검색어 테이블
- 순위분석 페이지 상단에 고정
- 5개 지표별 TOP10: 광고비, 노출수, 클릭수, 전환수, 매출액

### 4. 📊 TOTAL 행 순서 변경
- 새로운 순서:
  1. TOTAL (4주 평균)
  2. TOTAL (이전주 - 동적 주 이름)
  3. TOTAL (현재주 - 동적 주 이름) ← 증감율 표시

### 5. 💲 CPC 분석 필터 개선
- 효율분석과 동일한 필터 구조
- 추가된 필터: 제외 키워드, 캠페인 유형, 캠페인, 광고그룹
- 성능 기준: 최소 광고비/전환수/ROAS, 최대 전환당비용

### 6. 🔍 키워드 분석 개선
- ROAS 컬럼에 증감율 표시 추가

## 🚀 사용 방법

### 1단계: 파일 열기
```bash
# 웹 브라우저에서 파일 열기
marketing_report_v7.html
```

### 2단계: 데이터 업로드
1. "📤 데이터 업로드" 탭 선택
2. 4개 파일 업로드:
   - RAW.csv (일/주별 데이터)
   - RAW2.csv (검색어 데이터)
   - 소재.csv (상품 성과 데이터) - 새로 필요
   - 소재.tsv (상품 정보 매핑) - 새로 필요

### 3단계: 분석 확인
- 📈 성과 요약
- 🏆 순위 분석 (TOP10 포함)
- 🎯 캠페인 분석
- 🧩 광고그룹 분석
- 🔍 키워드 분석
- 🛍️ **상품 분석** ← 새 탭!
- 📰 매체별 분석
- ⚡ 효율분석
- 💲 CPC 분석 (개선된 필터)
- 📱 디바이스 분석

## 📁 필요한 데이터 파일

### 기존 파일 (그대로 사용)
1. **RAW.csv** - 일/주별 캠페인 데이터
2. **RAW2.csv** - 검색어 데이터

### 새로운 파일 (추가 필요)
3. **상품 성과 CSV** - 소재.csv와 동일한 형식
   ```
   캠페인유형,캠페인,광고그룹,상품ID,일별,주별,날짜,광고비,노출수,클릭수,전환수,매출액
   ```

4. **상품 TSV** - 소재.tsv와 동일한 형식
   ```
   [필드들...]  상품ID  [필드들...]  상품명  이미지URL
   ```
